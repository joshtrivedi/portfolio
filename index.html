My Work

Social Distancing Detector

A computer vision based machine learning project that determines whether the people are mantaining social distance by calculating the centroids of their respective rectangles, Click on the title to open the repository.

Virtual Whiteboard

Touché is a project in computer vision, that uses open-cv and media-pipe libraries for hand detection, and then allows the user to doodle on the screen based on his/her finger placement. Click on the title to open the repository.

Project Management System

Managing Projects can be hard for both Professors and Students so, This is a Software created in JavaFX and SQL that helps faculties assign projects and deadlines along with managing the Students enrollment data. Students also have access to a Login portal which tells them their assigned projects and deadlines.

Fingerprint Attendance Management System

Fingerprint Management System made using Arduino Uno 3 microcontroller along with a fingerprint scanner module which sends and stores the fingerprint data later on to be displayed in the .NET & C# application console with Mapping, Graphs, and Attendance Charts that are remotely sent over to the server.

Smart Agriculture Management System

Agriculture Management Tool that help farmers and contractors to check the conditions of the soil and field using the Arduino and sensors based tool that also uses a Java Frames GUI application displaying certain factors such as soil moisture, temparature, etc.

BoardBot - Discord Bot

A multifunction discord moderating bot made for the discord server for a youtuber to manage his server, BoardBot has functionalities such as: Muting/Banning/Timing out members, Automated responses, Role Responses and Emoji based roles (you send an emoji of a country flag and the bot maps it to the country tab and provides it to the user). Developed using the Open Source Javascript Library, Discord.JS and some basics of containers, A sprinkle of Regexes and some dazzle of JSON.

ChadBot - Discord Bot

Another multifunction discord bot made solelly for the purpose of experimenting Discord.JS updates and trying out new things, if you want to test out what this bot can do, please reach out to me and I shall add you in my discord server, where Chad can do what BoardBot can, and more! Unline it's predecessing bot brother, Chad has a seperate MongoDB instance, and was developed using cloudera and linux applications.
